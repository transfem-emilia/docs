## Introduction

Downloading a save backup for a Flash Game and Uploading it may be useful in the event that you may not keep access to the save stored by your browser.

## Downloading Saves

In order to download your saves to save them as a backup, use the following steps:

1. Right Click the Game Window, this is the square that the Flash Game is playing in.
2. Click the download button next to the title of the Flash Game

  In this example, the button circled would be used to download the user's save for Papa's Bakeria.

3. Your save file will be downloaded.

   Note: If your browser is setup to prompt you for where to store downloaded files, select where you would like the save file downloaded to.

## Uploading Saves

In order to upload your save backup, follow these steps:

1. Launch Flash Game as per normal.

   Note: If the Flash Game you are playing creates a save at launch, continue to step 3.
2. Create a save file in an empty slot, you do not need to play any levels for the Flash Game to create a save file.
3. Select the upload button next to the title of the Flash Game
